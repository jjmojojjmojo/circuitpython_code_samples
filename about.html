Source Code Viewer

This site contains source code used in a presentation, "State And Events In CircuitPython" originally presented to the Triangle Python User Group on 11/29/2018.

Each example is linked in the original slides. Note that, in most PDF viewers, the links in the above PDF file don't work - this code viewer relies on anchors to function and they aren't linked properly.
